## Weather App aka 'The Weathernator'

This was created to satisfy my curiousity about how other weather apps worked and where they imported the data from. I found out that the weather information was retrieved from a server with the use of an API and key to allow access. During this process I learned about working with asynchronous functions to retrieve data which was in JSON format and how to convert it into an object to populate my application. 

### Project Tools
  - Visual Studio Code
  - Accuweather API
  - HTML5
  - CSS3
  - Javascript ES6
  - Photoshop CC 2019
